# Я Бао Завари — frontend v2

Статический многостраничный frontend для GitHub Pages.

## Запуск
Откройте `index.html` или запустите локальный сервер: `python -m http.server 8000`.

## Перед публикацией замените
- `https://example.ru` в canonical, Open Graph, sitemap и schema;
- реквизиты оператора и e-mail в юридических документах;
- ориентировочные цены, длительность, вместимость и даты событий;
- демонстрационные изображения на реальные фото чайной;
- обработчик форм в `assets/js/components/forms.js`.

## Формы
Сейчас frontend валидирует поля и показывает успешное состояние. Для реальной отправки подключите серверный endpoint, CRM, Telegram-бота или WordPress.

## Источники фактических данных
Адрес, телефон, режим работы, меню и особенности сверялись с карточками 2ГИС и Яндекс Карт на дату сборки.
