# Hero-блок «Я Бао Завари» — Full HD video

## Что сделано

- В hero встроено исходное Full HD-видео 1920×1080.
- Видео оптимизировано **без повторного кодирования**: исходный H.264-поток перенесён из MOV в MP4 без изменения кадров.
- Добавлен `faststart`: индекс MP4 перенесён в начало файла, поэтому фон начинает загружаться раньше.
- Удалены лишние метаданные; аудиодорожки в исходнике не было.
- Первый кадр сохранён как качественный WebP-постер 1920×1080.
- Резкий стык 4,9-секундного ролика смягчается коротким переходом к точному первому кадру. Само видео при этом не перекодируется.
- Видео останавливается, когда hero уходит с экрана или вкладка неактивна.
- При `prefers-reduced-motion` и режиме экономии трафика показывается статичный постер.
- Параллакс и дополнительные CSS-анимации отключены: двигаются только пар и свеча внутри исходного ролика.

## Параметры видео

- разрешение: 1920×1080;
- кодек: H.264;
- цветовое пространство: BT.709;
- частота: около 24 FPS;
- длительность: 4,9 секунды;
- битрейт: около 6,95 Мбит/с;
- аудио: отсутствует;
- путь: `assets/video/hero-tea-loop.mp4`.

## Запуск

Из-за JavaScript-модулей проект лучше открывать через локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## GitHub Pages

Распакуйте содержимое архива в корень репозитория и включите GitHub Pages для ветки `main`.
