
# Я Бао Завари — frontend v4

Статическая версия для GitHub Pages и последующей интеграции в WordPress.

## Реализовано во второй волне
- исправлена адаптивная подача изображений в блоках с двухколоночной композицией;
- удалён блок атмосферных пространств;
- на главной меню показано категориями;
- слайдер отзывов поддерживает кнопки и колесо мыши, системный scrollbar скрыт;
- сглажен переход изображения и текста в CTA сертификатов;
- FAQ и юридические тексты используют всю ширину контейнера;
- добавлены хлебные крошки для меню, FAQ, отзывов и страницы события;
- возвращены фильтры меню с поддержкой query-параметра `?category=`;
- у всех событий отображается цена;
- внутреннее событие переработано: широкая обложка, основное описание и отдельная карточка записи;
- модальное окно событий автоматически получает название, дату, время и стоимость;
- обновлена карта с точкой организации;
- добавлен отступ между телефоном и иконками в подвале.

## Перед запуском
- заменить примерную ссылку Telegram;
- подключить реальную серверную отправку форм;
- уточнить даты, цены и тексты событий;
- заполнить юридические реквизиты.

## Динамическое наличие из Google Sheets

Страница `menu.html` получает из таблицы только публичные данные: название, категорию и наличие. Описание, изображение и цена остаются в карточках сайта и позже могут выводиться из WordPress.

Настройки находятся в `assets/js/config.catalog.js`:

- `spreadsheetId` — идентификатор таблицы;
- `sheets` — листы `Склад` и `Меню`;
- `outOfStockMode: 'mark'` — показывать карточку со статусом «Закончился»;
- `outOfStockMode: 'hide'` — скрывать закончившиеся и отсутствующие в таблице позиции;
- `cacheTtlMs` — срок кеширования данных в браузере.

Карточки сопоставляются по `data-stock-name` и `data-stock-aliases`. При переносе на WordPress эти значения следует заменить на отдельное поле связи с названием позиции в складской таблице.

Важно: при прямой загрузке из браузера адрес таблицы не отображается посетителю в интерфейсе, но технически доступен в исходном JavaScript и панели Network. Полностью скрыть его можно только через серверный endpoint WordPress.
