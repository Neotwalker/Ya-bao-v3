
# Я Бао Завари — frontend v3

Обновлённая статическая версия фронтенда для GitHub Pages и дальнейшей интеграции в WordPress.

## Что изменено
- убраны страницы `ceremonies.html` и `spaces.html`;
- три ключевых формата записи открывают модальное окно бронирования;
- в шапке добавлены телефон, VK и Telegram;
- убран список маркеров из hero;
- переработаны блоки событий, отзывов, FAQ и контактов;
- добавлены страницы `event-single.html` и `reviews.html`;
- обновлены изображения для блока первого визита, CTA и атмосферных карточек;
- структура подготовлена под лёгкую интеграцию с WordPress.

## Что заменить перед запуском
- ссылку на Telegram, если будет другой адрес;
- реальные тексты и даты событий;
- формирование отправки форм на сервер или в WordPress/CF7;
- юридические реквизиты в политике и согласии.
